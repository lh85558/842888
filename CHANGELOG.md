# 更新日志

## v1.0.0 (2023-12-01)
### ✨ 新功能
- 初始版本发布
- 集成CUPS 2.4.2打印服务
- 支持HP LaserJet 1020/1020plus/1007/1008/1108打印机
- 中文LuCI管理界面
- USB打印机即插即用
- 网络打印机支持
- 定时重启功能
- GitHub Actions云编译支持

### 🔧 技术特性
- 基于OpenWrt 21.02.x稳定分支
- 固件大小控制在16MB以内
- 支持TP842Nv3 (AR9531)硬件
- 优化的系统配置
- 完整的中文本地化

### 📋 默认配置
- LAN IP: 192.168.10.1
- WiFi SSID: THDN-dayin
- 管理账号: admin/thdn12345678
- 主机名: THDN-PrintServer

### 🎯 已知问题
- 首次启动可能需要1-2分钟完成初始化
- 某些USB3.0设备可能需要额外驱动

### 🚀 后续计划
- 添加更多打印机型号支持
- 优化Web界面用户体验
- 增加远程管理功能
- 支持云打印服务
